# Calculadora de IMC

Um projeto de estudo feito para treinar a escrita em HTML, o desing do CSS, e a lógica do JavaScript

## O que a calculadora faz?

O **IMC (Índice de Massa Corporal)** é um cálculo utilizado para avaliar se uma pessoa está dentro de uma faixa de peso considerada saudável em relação à sua altura.
Ele é obtido dividindo o **peso (kg)** pela **altura ao quadrado (m²)**. O resultado ajuda a classificar o peso em categorias como baixo peso, peso normal, sobrepeso ou obesidade.

Apesar de ser um cálcula muito usado ele pode ser pouco efetivo na maioria dos casos.
